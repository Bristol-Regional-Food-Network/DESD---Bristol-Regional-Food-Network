How the Hybrid Model Works

The recommendation model combines two machine learning techniques:

  • SVD (Matrix Factorisation) — identifies hidden patterns between users and products based on order history. Weighted at α = {{ summary.alpha|default:"0.6" }}.
  • Random Forest — predicts the probability a user will reorder a product, using features like category, frequency, and recency. Weighted at β = {{ summary.beta|default:"0.4" }}.

Final score = α × SVD score + β × RF probability
